### 表格属性

| Name        | Meaning                                        | Value               |
| ----------- | ---------------------------------------------- | ------------------- |
| border      | 表格边框，默认0，即无边框                      | 像素                |
| border-collapse      | 表格边框重合时的处理方法                    | collapse               |
| cellspacing | 单元格间距                                     | 像素，默认为2       |
| cellpadding | 单元格内容和单元格边框间距, `可以用来居中内容` | 像素，默认为1       |
| width       | 表格宽度                                       | 像素                |
| height      | 表格高度                                       | 像素                |
| align       | 表格作为`整体`在页面中的水平对齐方式           | left, center, right |
